# v6.15 — arm64-only APK (drop armeabi-v7a)

**Goal:** Shrink the release APK from ~37 MB to ~24 MB by shipping only the
`arm64-v8a` native libraries, dropping the 32-bit `armeabi-v7a` copies.

**Why:** The APK is dominated by ML Kit's per-ABI native `.so` blobs. A
universal APK carries BOTH ARM copies of `libtranslate_jni.so` (15.6 MB arm64 +
11.1 MB arm32) plus the language-id libs, even though any one phone runs only
one architecture. Andromeda targets modern devices only, distributed by
sideloading — every such phone is `arm64-v8a`. The 32-bit copy is pure dead
weight (~11.7 MB uncompressed).

## Design

One-line change to `abiFilters` in `app/build.gradle.kts`. No splits, no App
Bundle, no `bundletool` — a single universal APK is still produced, just
without the arm32 libraries. This keeps the entire build/stage/verify ritual
identical (one APK in, one APK out).

### `app/build.gradle.kts`
```
// before
ndk { abiFilters += listOf("arm64-v8a", "armeabi-v7a") }
// after
ndk { abiFilters += listOf("arm64-v8a") }
```
Also bump `versionCode 42 -> 43` and `versionName "6.14" -> "6.15"`.

## Consequences

- **arm32-only devices can no longer install the app** (installer reports "not
  compatible" — no crash, no partial install). Accepted: Andromeda targets
  modern arm64 devices only.
- **No functional change** on any installable device. Same features, same ML
  translation engine (arm64 lib does everything).
- `minSdk 26` is unaffected — ABI and SDK level are independent axes.
- Signing, dex-verify, md5-match staging rituals all unchanged (still one APK).
- Fully reversible: re-add `"armeabi-v7a"` and rebuild.

## Verification

- Clean release build succeeds.
- APK on-disk size measurably smaller than v6.14's 37,071,517 bytes (target
  ~24-28 MB; report the measured value).
- `unzip -l` on the APK shows `lib/arm64-v8a/libtranslate_jni.so` PRESENT and
  `lib/armeabi-v7a/` ABSENT.
- dex-verify: a unique v6.15-era string literal present in `classes.dex`.

## Non-goals

ABI splits (Option 2 — unnecessary when targeting a single architecture); App
Bundle / Play App Signing (Option 1 — deferred until Play distribution).
